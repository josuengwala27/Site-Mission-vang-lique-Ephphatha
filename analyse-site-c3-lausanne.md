# Analyse du site C3 Lausanne – Référence pour la Mission Évangélique Ephphatha

Site analysé : [C3 Lausanne](https://www.c3lausanne.ch/fr/index.html)  
Date d’analyse : mars 2025

---

## ⚠️ Important : forme vs. contenu

**Ce site C3 Lausanne est une référence pour la FORME uniquement** (structure des pages, types de rubriques, mise en page, idées de navigation). Votre pasteur l’a donné comme exemple de *ce qu’elle veut comme type de site*.

**La Mission Évangélique Ephphatha n’a pas le même programme ni le même fonctionnement.** Le contenu du futur site devra donc être **entièrement adapté à votre église** : vos horaires de culte, vos activités, vos groupes, vos lieux, vos projets, votre vision. Les exemples ci‑dessous (C3 Kids, Young Pros, Welcome Lunch, etc.) illustrent *le type* de pages possibles, pas un programme à copier. À vous de choisir quelles rubriques vous concernent et d’y mettre **votre** contenu.

- **À s’inspirer de C3 :** structure du site, types de pages, organisation du menu, idées de formulaires et de liens.
- **À faire spécifique à Ephphatha :** tout le texte, le programme, les activités, les horaires, les lieux, les noms, la vision.

---

## 1. Vue d’ensemble (C3 – pour illustrer la forme)

C3 Lausanne est une église bilingue (FR/EN) à Lausanne. Le site présente l’église, les cultes, les activités en semaine, le podcast, le calendrier, les dons et les moyens de s’impliquer. Il sert à la fois de vitrine et d’outil pratique (inscriptions, dons, demandes de prière, culte en ligne). **Pour Ephphatha, on garde l’idée de “vitrine + outils pratiques”, avec VOTRE programme et VOS infos.**

## 4. Résumé pour la Mission Évangélique Ephphatha

**Rappel :** Les lignes ci‑dessous décrivent des *types* de pages et des bonnes pratiques de forme (inspirées de C3). Le contenu de chaque page sera **celui de votre église** — programme, horaires, activités, vision — pas une copie de C3.

### 4.1 Types de pages à prévoir (forme inspirée de C3, contenu = Ephphatha)

| Priorité | Type de page | Idée (forme) — à remplir avec VOS infos |
|----------|--------------|----------------------------------------|
| **Haute** | Accueil | Slogan, courte présentation de votre église, éventuelle vidéo du pasteur, liens rapides (prière, Jésus, donner, contact). |
| **Haute** | Culte / Réunions | *Vos* lieu(x), *vos* horaires, programme enfants si vous en avez, livestream si vous le faites. |
| **Haute** | Contact | *Vos* adresse(s), tél/WhatsApp, email, formulaire, liens prière / technique si besoin. |
| **Haute** | Donner | Texte sur les dons + *votre* virement (IBAN, communication) + don en ligne / espèces selon ce que vous proposez. |
| **Haute** | En savoir plus sur Jésus | Qui est Jésus, accepter Jésus (prière), « Et ensuite ? » avec *votre* contact. |
| **Moyenne** | Calendrier / Événements | *Vos* cultes, *vos* réunions, *vos* événements (contenu différent de C3). |
| **Moyenne** | Podcast / Messages | Si vous enregistrez : *vos* messages (YouTube ou autre). |
| **Moyenne** | Enfants / Jeunesse | Uniquement si vous avez ce type d’activité : *vos* groupes, âges, horaires, lieu. |
| **Basse** | Groupes / Semaine | Uniquement si pertinent pour vous : *vos* groupes, études bibliques, activités. |
| **Basse** | Baptêmes / Soirées spéciales | Si vous en organisez : *vos* dates et infos. |
| **Optionnel** | Accueil des nouveaux | Si vous faites un événement dédié : *votre* formule. |
| **Optionnel** | Aide / Counseling | Si vous proposez un accompagnement : *votre* offre. |

### 4.2 Bonnes pratiques de forme (à adapter, pas à copier le contenu)

1. **Bloc de liens en bas de chaque page** : prière, aide technique, Jésus, donner – très utile pour les visiteurs.  
2. **Une page « Jésus »** claire : évangélisation et suivi (email, livre, Bible).  
3. **Contact multiple** : formulaire + email + téléphone/WhatsApp.  
4. **Donner** : plusieurs options (virement, en ligne, espèces) + message sur l’impact des dons.  
5. **Calendrier** : un seul endroit pour voir tous les événements, avec liens vers les détails.  
6. **Formulaires** : confidentialité + captcha sur tous les formulaires.  
7. **Multilingue** : si votre église est bilingue, structure FR/EN comme C3 peut être un bon modèle.  
8. **Réseaux sociaux et app** : liens visibles (footer ou header) pour rester en contact.
